1. Построение модели данных html через js

a) выносим html блоки-сообщения в константу model в структуре тип, значение, опции
b) весь контернт обварачиваем в html и далее внедряем через document.querySelector(...) в insertAdjacentHTML в главную страницу index.html
c) теперь создана единая страница "Model", где возможно  легко менять контент и инлайн стили

2. OOP
- строим интерфейс блок
- переносим все объекты Model в конструктор класса Блок и 1м методом toHTML
- создаем к-во блоков-наследников от Блок и переопределенным toHTML каждого из блоков
- имплементируем каждый из объектов в Model
- каждый блок можно вызывать единым методом toHTML родителя Блок, таким образом нет необходимости применять if
для распознавания объекта и функций обработки.
